freeCodeCamp Project: Image Search Abstraction Layer Microservice

User stories:

  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  2. I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  3. I can get a list of the most recently submitted search strings.

Example of listing images by keyword

Entering:

      https://image-search-microservice-pa.glitch.me/api/imagesearch/gorilla

will give something like:

      [
        {
          "species": "Gorilla gorilla",
          "status": "critically endangered",
          "thumbnail": "https://upload.wikimedia.org/wikipedia/commons/thumb/5/50/Male_gorilla_in_SF_zoo.jpg/220px-Male_gorilla_in_SF_zoo.jpg",
          "context": "https://en.wikipedia.org/wiki/Gorilla",
          "alt-text": "Western gorillas are ground dwelling herbivorous apes from central Sub-Saharan Africa"
        },
        {
          "species": "Gorilla beringei",
          "status": "critically endangered",
          "thumbnail": "https://upload.wikimedia.org/wikipedia/commons/thumb/9/98/Mountain_gorillas_%288209001529%29.jpg/220px-Mountain_gorillas_%288209001529%29.jpg",
          "context": "https://en.wikipedia.org/wiki/Eastern_gorilla",
          "alt-text": "Eastern Gorillas are the largest living primate"
        }
      ]

The meta-data of at most 10 images are listed: use ?offset= to list yet more meta-data.

Example of listing the latest queries

Entering:

      https://image-search-microservice-pa.glitch.me/api/latest/imagesearch/

will give something like:

      {
        "find": "gorilla",
        "found": 2,
        "stamp": "Sun, 18 Mar 2018 17:31:23 GMT"
      },

Note the queries are presented newest first, oldest last. Only the last 10 are listed: use ?offset= to see older queries.

Keywords

The following keywords should retrieve meta-data for an image or two:

  • lemur
  • tarsier
  • marmoset
  • spider monkey
  • swamp monkey
  • gibbon
  • gorilla
  • chimpanzee
  • orangutan

The following keywords may retrieve enough meta-data to make the `?offset=` worth using.

  • new world monkeys
  • old world monkeys
  • monkeys
  • great apes

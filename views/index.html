freeCodeCamp Project: File Metadata Microservice

User stories:

  1. I can submit a FormData object that includes a file upload.
  2. I will receive the file size in bytes in the JSON response.

Example:

  • Click on the Browse... button below and select file.
  • Click on the Submit button to upload the file to calculate its size.
  • The JSON response will look something like:
      {
        "file": "fizzbuzz.erl",
        "size": 211
      }

Upload a file to view its size in bytes.
